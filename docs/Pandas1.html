14.1.2 使用可调用函数索引

.loc, .iloc 和 []可以接受可调用的函数作为索引。可调用函数是接收一个参数(调用它的series或dataframe)的函数，返回有效输出进行索引。

import pandas as pd
import numpy as np
mydf = pd.DataFrame(np.random.randn(6, 4),
    index=list('abcdef'),
    columns=list('ABCD'))
mydf
##           A         B         C         D
## a  0.227897 -0.709347 -0.383076  1.052497
## b  1.235967  0.378036  0.964515  0.327272
## c -0.237934 -0.291498 -0.575182  0.896539
## d -0.948367 -0.063022  0.888085 -0.800577
## e  0.623064 -0.668269 -0.283534 -0.092795
## f -0.521646 -1.434531 -1.337873 -0.715154
mydf.loc[lambda df: df['A'] > 0, :]
##           A         B         C         D
## a  0.227897 -0.709347 -0.383076  1.052497
## b  1.235967  0.378036  0.964515  0.327272
## e  0.623064 -0.668269 -0.283534 -0.092795
mydf.loc[:, lambda df: ['A', 'B']]
##           A         B
## a  0.227897 -0.709347
## b  1.235967  0.378036
## c -0.237934 -0.291498
## d -0.948367 -0.063022
## e  0.623064 -0.668269
## f -0.521646 -1.434531
mydf[lambda df: df.columns[0]]
## a    0.227897
## b    1.235967
## c   -0.237934
## d   -0.948367
## e    0.623064

14.1.3 随机抽样

默认无放回抽样

s = pd.Series([0, 1, 2, 3, 4, 5])
s.sample(n=3)
## 3    3
## 1    1
## 5    5
## dtype: int64
s.sample(frac=0.5)
## 0    0
## 3    3
## 2    2
## dtype: int64

可以为不同的值分配不同的权重：

s.sample(n=3, weights = [0, 0, 0.2, 0.2, 0.2, 0.4]
)
## 5    5
## 2    2
## 3    3
## dtype: int64
mydf.sample(n=3)

14.1.6 isin 索引

isin用于判断元素是否在指定范围中，返回逻辑向量。

s = pd.Series(np.arange(5), index=np.arange(5)[::-1], dtype='int64')
s.isin([2, 4, 6])
## 4    False
## 3    False
## 2     True
## 1    False
## 0     True
## dtype: bool
s[s.isin([2, 4, 6])]
## 2    2
## 0    4
## dtype: int64
s[s.index.isin([2, 4, 6])]
## 4    0
## 2    2
## dtype: int64

对于DataFrame也可以使用。

df = pd.DataFrame({'vals': [1, 2, 3, 4], 'ids': ['a', 'b', 'f', 'n'],
                   'ids2': ['a', 'n', 'c', 'n']})
values = ['b', 1]
df.isin(values)
##     vals    ids   ids2
## 0   True  False  False
## 1  False   True  False
## 2  False  False  False
## 3  False  False  False
# 返回行逻辑索引：如果该行有元素满足给定条件则返回True
# row_mask = df.isin(values).any(1)
df[df.isin(values).any(1)]
##    vals ids ids2
## 0     1   a    a
## 1     2   b    n
# 返回列逻辑索引：如果该列有元素满足给定条件则返回True
# col_mask = df.isin(values).any(0)
df.loc[:, df.isin(values).any(0)]
##    vals ids
## 0     1   a
## 1     2   b
## 2     3   f
## 3     4   n

对于DataFrame可以指定每列需满足的条件。

values = {'ids': ['a', 'b'], 'vals': [1, 3]}
df.isin(values)
##     vals    ids   ids2
## 0   True   True  False
## 1  False   True  False
## 2   True  False  False
## 3  False  False  False
row_mask = df.isin(values).all(1)
df[row_mask]
## Empty DataFrame
## Columns: [vals, ids, ids2]
## Index: []

14.1.8 使用numpy()有条件地数据扩展

df = pd.DataFrame({'col1': list('ABBC'), 'col2': list('ZZXY')})
df['color'] = np.where(df['col2'] == 'Z', 'green', 'red')
df
##   col1 col2  color
## 0    A    Z  green
## 1    B    Z  green
## 2    B    X    red
## 3    C    Y    red

如果有多个逻辑判断条件，可以使用numpy.select()。

conditions = [
    (df['col2'] == 'Z') & (df['col1'] == 'A'),
    (df['col2'] == 'Z') & (df['col1'] == 'B'),
    (df['col1'] == 'B')
]
choices = ['yellow', 'blue', 'purple']
df['color'] = np.select(conditions, choices, default='black')
df
##   col1 col2   color
## 0    A    Z  yellow
## 1    B    Z    blue
## 2    B    X  purple
## 3    C    Y   black

14.1.9 query() 方法

DataFrame对象具有query()方法，允许使用表达式选择。

df = pd.DataFrame(np.random.randint(1, 10, size = [5, 3]), columns=list('abc'))
df.query('(a < b) & (b < c)')
# df.query('a < b < c')
# df[(df['a'] < df['b']) & (df['b'] < df['c'])]
##    a  b  c
## 2  4  7  8

对于Index也能使用：

df.query('index < b < c')
# 或者
# df.index.name = 'x'
# df.query('a < x and b < c')

14.2 重复数据

Two methods takes as an argument the columns to use to identify duplicated rows.

  • duplicated returns a boolean vector whose length is the number of rows, and which indicates whether a row is duplicated.
  • drop_duplicates removes duplicate rows.

Each method has a keep parameter to specify targets to be kept.

  • keep='first' (default): mark / drop duplicates except for the first occurrence.
  • keep='last': mark / drop duplicates except for the last occurrence.
  • keep=False: mark / drop all duplicates.
df2 = pd.DataFrame({'a': ['one', 'one', 'two', 'two', 'two', 'three', 'four'],
                    'b': ['x', 'y', 'x', 'y', 'x', 'x', 'x'],
                    'c': np.random.randn(7)},
                    index=['a', 'a', 'b', 'c', 'b', 'a', 'c'])
df2

14.3.2 执行顺序很重要

以下是使用.loc对多个项目（使用mask）和单个项目使用固定索引的推荐方法：

dfc = pd.DataFrame({'a': ['one', 'one', 'two',
                          'three', 'two', 'one', 'six'],
                    'c': np.arange(7)})
dfd = dfc.copy()

# Setting multiple items using a mask
mask = dfd['a'].str.startswith('o')
dfd.loc[mask, 'c'] = 42
dfd
##        a   c
## 0    one  42
## 1    one  42
## 2    two   2
## 3  three   3
## 4    two   4
## 5    one  42
## 6    six   6
# Setting a single item
dfd = dfc.copy()
dfd.loc[2, 'a'] = 11
dfd
##        a  c
## 0    one  0
## 1    one  1
## 2     11  2
## 3  three  3
## 4    two  4
## 5    one  5
## 6    six  6
